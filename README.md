# Airbnb Price Prediction

This project predicts Airbnb listing prices using machine learning.

## Dataset
AB_NYC_2019.csv (New York Airbnb listings)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## How to Run
1. Create virtual environment
2. Install requirements
3. Run:
   python airbnb_price_prediction.py
